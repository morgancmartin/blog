Archlinux

June 8, 2015

Okay, so it’s been a minute. Let me get you caught up:

  • I postponed my chess project. I got everything working minus the FINAL method, "checkmate?" It’s surprisingly difficult. I decided to move onto Rails. I intend to come back to this project someday and knock it out of the park
  • So far Rails has been fun. MVC and all that.
  • Real life got in the way and I had to switch laptops. I also got behind on coding for a few days.
  • I decided to install Arch Linux onto my new Lenovo T420 laptop. I’ve never run Linux before, so Arch has definitely been a shocker for me.

So, if you know anything about Arch Linux, you’ll know that it’s about as barebones as it gets when it comes to Operating Systems. I installed it because I’d read it’s a good OS for performance, and I’ve been thinking about making the switch to Linux for a while now. Arch was definitely like jumping into the deep end of the pool, but I feel like I’ve learned a lot in the past two days it’s taken to set it all up.

Chess

May 19, 2015

Okay, so chess isn’t going to take a week. I started working on it yesterday, and despite not having much time for it today, I’m already getting pretty close. I’ve got my board class and piece class mostly setup and I’m beginning on my game class now, so I should have it finished tomorrow if all goes well.

Chess is really interesting/fun! I’ve made a point of not cheating off anyone else’s code, so that I can be sure that I’m ready for Rails. Then again, I did install a neat gem called Colored, that just helps me color a given string in the command line. I could have done that on my own, but it looked a little tedious, plus gems are still part of the whole thing anyway, maaaaann.

Oh, and I also caught myself up on Git and all of it’s features. It’s still got a lot of mystical-seeming functionality, but I’ve got a much better understanding of what it’s doing, and why it’s important, so I’m happy with my bare-bones knowledge for now.

That’s about it!
